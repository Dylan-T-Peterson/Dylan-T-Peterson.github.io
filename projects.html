Tableau Superstore Dataset

Tools Used

  • Python
    • Numpy
    • Pandas
    • Matplotlib
    • Seaborn
  • Tableau

This project uses the Tableau Superstore Dataset and portrays my capabilities with Tableau and Python with GitHub. Using calculated fields created with each program's tools, I could reliably group, analyze, and visualize the data. This project showed me my preference for Python for properly aggregating and calculating data for analysis or visualization. I found it easier to program the needed fields using Pandas' library and SIMD instructions over Tableau's bespoke and Excel-like function calls and syntax. I will likely use Tableau for the story and dashboard features. Still, I will likely lean closer to slideshow applications (i.e., Google Sheets and Microsoft PowerPoint) over the story feature, as that application genre, combined with customized Seaborn visualizations, has the potential for a more personalized and unique presentation.

Sources: Tableau dataset

Financial Database

Tools Used

  • Python
    • SQLite3
    • Pandas
  • SQLite3

This project is to show my capabilities in creating relational databases and necessary personalized APIs to curate how the database interacts. The Python scripts are complete and built-in with type verification and guard clauses to ensure that any input gained from the user is of a valid type/format and will not pose a problem for the database.

I started by thinking about the relational database structure I'd need for the project, whether the specific columns would be necessary for operation, and which ones I would need for calculating back-end fields. After charting the database, I created the initialization SQL script that would be modified as required while making a CLI interface and the necessary back-end APIs. I then referred to my previously charted interactions between the database and the Python program to see in which order it would be the most logical/necessary to move on to the following function.
